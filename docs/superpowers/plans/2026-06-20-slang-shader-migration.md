# Shader-Slang Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate all 22 hand-written GLSL shaders to Shader-Slang, compiling to optimized GLSL 330 at build time, with shared includes, an `ILight` interface + generics for the PBR pipeline, and HLSL register syntax throughout.

**Architecture:** Slang source files live in `assets/shaders/slang/`; slangc (from vcpkg) is invoked as a CMake custom command per entry point, emitting GLSL files into `${CMAKE_BINARY_DIR}/shaders/glsl/`. The existing game CMakeLists post-build step is extended to copy those compiled GLSL files into `$<TARGET_FILE_DIR:game>/assets/shaders/glsl/` alongside other assets. The C++ `Shader` loader is unchanged — it still reads `.glsl` files by name; they are now generated rather than hand-authored.

**Tech Stack:** Shader-Slang (slangc via vcpkg), CMake custom commands, OpenGL GLSL 330, existing sponge renderer C++.

## Global Constraints

* Target language: GLSL 330 core (`-target glsl -profile glsl_330`)
* All Slang sources go in `assets/shaders/slang/` (new directory)
* Output GLSL files must land at runtime path `assets/shaders/glsl/<name>` — the path the C++ `Shader` loader already uses
* Use HLSL `register(tN)` / `register(sN)` syntax for all texture/sampler bindings
* Slang generics and interfaces are resolved at compile time — no runtime dispatch
* Do not edit files under `sponge/deps/` or `out/`
* Do not modify the C++ `Shader` class or uniform-setting calls — plain GLSL `uniform` globals in Slang map 1-to-1 to what `glUniform*` already sets
* Commit after each task; build must pass before proceeding to the next task

* \[ ] **Step 1: Add slangc discovery and helper function to root CMakeLists.txt**

```cmake
# --- Shader-Slang compilation ---

find_program(SLANGC_EXECUTABLE slangc
    REQUIRED
    HINTS
        "${CMAKE_SOURCE_DIR}/vcpkg_installed/x64-windows/tools/shader-slang"
        "${CMAKE_SOURCE_DIR}/vcpkg_installed/x64-linux/tools/shader-slang"
        "${CMAKE_SOURCE_DIR}/vcpkg_installed/arm64-osx/tools/shader-slang"
)
message(STATUS "Found slangc: ${SLANGC_EXECUTABLE}")

set(SLANG_SOURCE_DIR "${CMAKE_SOURCE_DIR}/assets/shaders/slang")
set(SLANG_OUTPUT_DIR "${CMAKE_BINARY_DIR}/shaders/glsl")
file(MAKE_DIRECTORY "${SLANG_OUTPUT_DIR}")

# Usage: compile_slang_shader(<slang-file> <entry-point> <output-glsl-name>)
# Appends the output path to SLANG_COMPILED_OUTPUTS (parent scope).
function(compile_slang_shader SLANG_FILE ENTRY_POINT OUTPUT_NAME)
    set(INPUT  "${SLANG_SOURCE_DIR}/${SLANG_FILE}")
    set(OUTPUT "${SLANG_OUTPUT_DIR}/${OUTPUT_NAME}")
    add_custom_command(
        OUTPUT  "${OUTPUT}"
        COMMAND "${SLANGC_EXECUTABLE}"
                "${INPUT}"
                -entry "${ENTRY_POINT}"
                -target glsl
                -profile glsl_330
                -o "${OUTPUT}"
        DEPENDS "${INPUT}"
        COMMENT "slangc: ${SLANG_FILE} [${ENTRY_POINT}] -> ${OUTPUT_NAME}"
        VERBATIM
    )
    set(SLANG_COMPILED_OUTPUTS "${SLANG_COMPILED_OUTPUTS};${OUTPUT}" PARENT_SCOPE)
endfunction()

set(SLANG_COMPILED_OUTPUTS "")

# screenquad (shared vertex for blur and fxaa passes)
compile_slang_shader(screenquad.vert.slang     vertMain   blur.vert.glsl)
compile_slang_shader(screenquad.vert.slang     vertMain   fxaa.vert.glsl)
```

```slang
#pragma once
#include "common.slang"

// ---------------------------------------------------------------------------
// Light interface — all light types conform to this.
// evaluatePBR<L:ILight>() is monomorphized per concrete type at compile time.
// ---------------------------------------------------------------------------
interface ILight
{
    // World-space unit vector from worldPos toward the light.
    float3 getDirection(float3 worldPos);

    // Pre-attenuated radiance arriving at worldPos.
    float3 getRadiance(float3 worldPos);
}

// ---------------------------------------------------------------------------
// Concrete light types
// ---------------------------------------------------------------------------

struct DirectionalLight
{
    bool   enabled;
    bool   castShadow;
    float3 color;
    float3 direction;
}

extension DirectionalLight : ILight
{
    float3 getDirection(float3 worldPos) { return normalize(-direction); }
    float3 getRadiance(float3 worldPos)  { return color; }
}

struct PointLight
{
    float3 color;
    float3 position;
    int    attenuationIndex;
}

extension PointLight : ILight
{
    float3 getDirection(float3 worldPos) { return normalize(position - worldPos); }
    float3 getRadiance(float3 worldPos)
    {
        float  dist = length(position - worldPos);
        int    idx  = clamp(attenuationIndex, 0, 10);
        float3 att  = kLightAttenuation[idx];
        float  atten = 1.0 / (att.x + att.y * dist + att.z * dist * dist);
        return color * atten;
    }
}

// ---------------------------------------------------------------------------
// PBR BRDF helpers
// ---------------------------------------------------------------------------

float3 fresnelSchlick(float cosTheta, float3 F0)
{
    // Optimized exponent — Karis 2013
    return F0 + (1.0 - F0) * pow(2.0, (-5.55473 * cosTheta - 6.98316) * cosTheta);
}

float distributionGGX(float3 N, float3 H, float rough)
{
    float a      = rough * rough;
    float a2     = a * a;
    float NdotH  = max(dot(N, H), 0.0);
    float denom  = (NdotH * NdotH * (a2 - 1.0) + 1.0);
    return a2 / (M_PI * denom * denom);
}

float geometrySchlickGGX(float NdotV, float rough)
{
    float r = rough + 1.0;
    float k = (r * r) / 8.0;
    return NdotV / (NdotV * (1.0 - k) + k);
}

float geometrySmith(float NdotV, float NdotL, float rough)
{
    return geometrySchlickGGX(NdotV, rough) * geometrySchlickGGX(NdotL, rough);
}

// ---------------------------------------------------------------------------
// Generic PBR evaluation — resolved at compile time per light type.
// ---------------------------------------------------------------------------
float3 evaluatePBR<L : ILight>(
    L      light,
    float3 worldPos,
    float3 albedo,
    float3 N,
    float3 V,
    float  metallic,
    float  roughness)
{
    float3 Ldir     = light.getDirection(worldPos);
    float3 radiance = light.getRadiance(worldPos);
    float3 H        = normalize(V + Ldir);
    float  NdotV    = max(dot(N, V),    1e-5);
    float  NdotL    = max(dot(N, Ldir), 1e-5);
    float3 F0       = lerp(float3(0.04), albedo, metallic);
    float  NDF      = distributionGGX(N, H, roughness);
    float  G        = geometrySmith(NdotV, NdotL, roughness);
    float3 F        = fresnelSchlick(max(dot(H, V), 0.0), F0);
    float3 specular = (NDF * G * F) / (4.0 * NdotV * NdotL + 1e-5);
    float3 kD       = (float3(1.0) - F) * (1.0 - metallic);
    return (kD * albedo / M_PI + specular) * radiance * NdotL;
}
```

```slang
#pragma once

// EVSM shadow lookup. Returns shadow factor in [0,1] where 1 = fully shadowed.
float shadowEVSM(
    Texture2D<float2> shadowMap,
    SamplerState      shadowSampler,
    float4            fragPosLightSpace,
    float             evsmBleedThreshold)
{
    float3 proj = fragPosLightSpace.xyz / fragPosLightSpace.w;
    proj = proj * 0.5 + 0.5;
    if (proj.z > 1.0 || proj.z < 0.0)
        return 0.0;

    float  currentDepth = proj.z;
    float2 moments      = shadowMap.Sample(shadowSampler, proj.xy);

    float p        = float(currentDepth <= moments.x);
    float variance = max(moments.y - moments.x * moments.x, 0.00002);
    float d        = currentDepth - moments.x;
    float pMax     = variance / (variance + d * d);
    pMax = clamp((pMax - evsmBleedThreshold) / (1.0 - evsmBleedThreshold), 0.0, 1.0);
    return 1.0 - max(p, pMax);
}
```

```slang
// pbr.vert.slang
uniform float4x4 mvp;
uniform float4x4 model;
uniform float4x4 lightSpaceMatrix;

struct VSInput
{
    float3 position : POSITION;
    float2 texCoord : TEXCOORD0;
    float3 normal   : NORMAL;
}

struct VSOutput
{
    float4 pos              : SV_Position;
    float3 worldPosition    : TEXCOORD0;
    float3 worldNormal      : TEXCOORD1;
    float2 texCoords        : TEXCOORD2;
    float4 fragPosLightSpace: TEXCOORD3;
}

[shader("vertex")]
VSOutput vertMain(VSInput input)
{
    VSOutput output;
    float4 worldPos4        = mul(model, float4(input.position, 1.0));
    output.pos              = mul(mvp,   float4(input.position, 1.0));
    output.worldPosition    = worldPos4.xyz;
    // Normal matrix = transpose(inverse(model)) — computed on CPU ideally,
    // but reproduced here to match original shader exactly.
    float3x3 normalMatrix   = transpose(inverse((float3x3)model));
    output.worldNormal      = mul(normalMatrix, input.normal);
    output.texCoords        = input.texCoord;
    output.fragPosLightSpace= mul(lightSpaceMatrix, worldPos4);
    return output;
}
```

```slang
    float4 pos              : SV_Position;
    float3 worldPosition    : TEXCOORD0;
    float3 worldNormal      : TEXCOORD1;
    float2 texCoords        : TEXCOORD2;
    float4 fragPosLightSpace: TEXCOORD3;
}

[shader("fragment")]
float4 fragMain(FSInput input) : SV_Target
{
    float3 texColor   = pow(texture_diffuse1.Sample(sampler_diffuse1, input.texCoords).rgb, float3(2.2));
    float3 constColor = pow(float3(1.0), float3(2.2));
    float3 albedo     = lerp(texColor, constColor, float(hasNoTexture));

    float3 N  = normalize(input.worldNormal);
    float3 V  = normalize(viewPos - input.worldPosition);
    float3 Lo = float3(0.0);

    // Directional light — evaluatePBR<DirectionalLight> specialized at compile time
    float3 dirContrib = evaluatePBR(directionalLight, input.worldPosition, albedo, N, V, metallic, roughness);
    float  shadow     = 0.0;
    if (directionalLight.castShadow)
        shadow = shadowEVSM(shadowMap, sampler_shadowMap, input.fragPosLightSpace, evsmBleedThreshold);
    Lo += dirContrib * (1.0 - shadow) * float(directionalLight.enabled);

    // Point lights — evaluatePBR<PointLight> specialized at compile time
    for (int i = 0; i < numLights; ++i)
        Lo += evaluatePBR(pointLights[i], input.worldPosition, albedo, N, V, metallic, roughness);

    float3 ambient = float3(ambientStrength) * albedo * ao;
    float3 color   = ambient + Lo;

    // Reinhard HDR tone mapping
    color = color / (color + float3(1.0));

    // Gamma correction
    color = pow(color, float3(1.0 / 2.2));

    return float4(color + emissive, 1.0);
}
```

## Task 7: Migrate FXAA shader

The `fxaa.frag.glsl` combines FXAA 3.11 with bloom overlay. This shader uses `textureOffset` which maps to `SampleOffset` in Slang/HLSL.

**Interfaces:**

* \[ ] **Step 1: Create fxaa.frag.slang**

```slang
// fxaa.frag.slang — FXAA 3.11 (quality preset 12) + bloom overlay
// Based on Timothy Lottes / NVIDIA, adapted to Slang.

Texture2D<float4> screenTexture       : register(t0);
SamplerState      screenSampler       : register(s0);
Texture2D<float4> bloomTex            : register(t1);
SamplerState      bloomSampler        : register(s1);
uniform float2    rcpFrame;
uniform float     bloomIntensity;

// Quality preset 12: 5-step search (1.5, 2.0, 2.0, 2.0, 8.0)
static const float FXAA_SUBPIX             = 0.75;
static const float FXAA_EDGE_THRESHOLD     = 0.125;
static const float FXAA_EDGE_THRESHOLD_MIN = 0.0625;
static const float kSteps[5]               = { 1.5, 2.0, 2.0, 2.0, 8.0 };

// Green-channel-weighted luma (Lottes formula — blue excluded)
float fxaaLuma(float3 rgb) { return rgb.y * (0.587 / 0.299) + rgb.x; }

float4 fxaaPass(float2 uv)
{
    float4 rgbaM = screenTexture.Sample(screenSampler, uv);
    float  lumaM = fxaaLuma(rgbaM.rgb);

    float lumaN  = fxaaLuma(screenTexture.SampleOffset(screenSampler, uv, int2( 0,  1)).rgb);
    float lumaS  = fxaaLuma(screenTexture.SampleOffset(screenSampler, uv, int2( 0, -1)).rgb);
    float lumaE  = fxaaLuma(screenTexture.SampleOffset(screenSampler, uv, int2( 1,  0)).rgb);
    float lumaW  = fxaaLuma(screenTexture.SampleOffset(screenSampler, uv, int2(-1,  0)).rgb);

    float rangeMax = max(lumaM, max(max(lumaN, lumaS), max(lumaE, lumaW)));
    float rangeMin = min(lumaM, min(min(lumaN, lumaS), min(lumaE, lumaW)));
    float range    = rangeMax - rangeMin;

    if (range < max(FXAA_EDGE_THRESHOLD_MIN, rangeMax * FXAA_EDGE_THRESHOLD))
        return rgbaM;

    float lumaNW = fxaaLuma(screenTexture.SampleOffset(screenSampler, uv, int2(-1,  1)).rgb);
    float lumaNE = fxaaLuma(screenTexture.SampleOffset(screenSampler, uv, int2( 1,  1)).rgb);
    float lumaSW = fxaaLuma(screenTexture.SampleOffset(screenSampler, uv, int2(-1, -1)).rgb);
    float lumaSE = fxaaLuma(screenTexture.SampleOffset(screenSampler, uv, int2( 1, -1)).rgb);

    float subpixNSEW = lumaN + lumaS + lumaE + lumaW;
    float subpixA    = (2.0 * subpixNSEW + lumaNW + lumaNE + lumaSW + lumaSE) / 12.0;
    float subpixB    = clamp(abs(subpixA - lumaM) / range, 0.0, 1.0);
    float subpixC    = (-2.0 * subpixB + 3.0) * subpixB * subpixB;
    float subpixD    = subpixC * subpixC * FXAA_SUBPIX;

    float edgeHorz =
        abs(lumaNW + -2.0 * lumaN + lumaNE) +
        abs(lumaW  + -2.0 * lumaM + lumaE ) * 2.0 +
        abs(lumaSW + -2.0 * lumaS + lumaSE);
    float edgeVert =
        abs(lumaNW + -2.0 * lumaW + lumaSW) +
        abs(lumaN  + -2.0 * lumaM + lumaS ) * 2.0 +
        abs(lumaNE + -2.0 * lumaE + lumaSE);
    bool horzSpan = (edgeHorz >= edgeVert);

    float luma1 = horzSpan ? lumaN : lumaW;
    float luma2 = horzSpan ? lumaS : lumaE;
    float grad1 = luma1 - lumaM;
    float grad2 = luma2 - lumaM;
    bool  pairN = abs(grad1) >= abs(grad2);

    float gradientScaled = 0.25 * max(abs(grad1), abs(grad2));
    float stepLen        = horzSpan ? rcpFrame.y : rcpFrame.x;
    if (pairN) stepLen   = -stepLen;
    float localAvg       = 0.5 * (pairN ? luma1 : luma2) + 0.5 * lumaM;

    float2 posP  = uv;
    if (horzSpan) posP.y += stepLen * 0.5;
    else          posP.x += stepLen * 0.5;

    float2 offNP = horzSpan ? float2(rcpFrame.x, 0.0) : float2(0.0, rcpFrame.y);

    float2 posN      = posP - offNP * kSteps[0];
    float2 posPos    = posP + offNP * kSteps[0];
    float  lumaEndN  = fxaaLuma(screenTexture.Sample(screenSampler, posN  ).rgb) - localAvg;
    float  lumaEndP  = fxaaLuma(screenTexture.Sample(screenSampler, posPos).rgb) - localAvg;
    bool   doneN     = abs(lumaEndN) >= gradientScaled;
    bool   doneP     = abs(lumaEndP) >= gradientScaled;

    for (int i = 1; i < 5; i++)
    {
        if (!doneN) {
            posN    -= offNP * kSteps[i];
            lumaEndN = fxaaLuma(screenTexture.Sample(screenSampler, posN  ).rgb) - localAvg;
            doneN    = abs(lumaEndN) >= gradientScaled;
        }
        if (!doneP) {
            posPos  += offNP * kSteps[i];
            lumaEndP = fxaaLuma(screenTexture.Sample(screenSampler, posPos).rgb) - localAvg;
            doneP    = abs(lumaEndP) >= gradientScaled;
        }
        if (doneN && doneP) break;
    }

    float dstN      = horzSpan ? (uv.x - posN.x  ) : (uv.y - posN.y  );
    float dstP      = horzSpan ? (posPos.x - uv.x ) : (posPos.y - uv.y);
    bool  goodSpanN = (lumaEndN < 0.0) != (lumaM < localAvg);
    bool  goodSpanP = (lumaEndP < 0.0) != (lumaM < localAvg);
    float spanLen   = dstN + dstP;

    float pixelOffset = max(
        goodSpanN ? (0.5 - dstN / spanLen) : 0.0,
        goodSpanP ? (0.5 - dstP / spanLen) : 0.0
    );
    pixelOffset = max(pixelOffset, subpixD);

    float2 finalUV = uv;
    if (horzSpan) finalUV.y += pixelOffset * stepLen;
    else          finalUV.x += pixelOffset * stepLen;

    return screenTexture.Sample(screenSampler, finalUV);
}

struct FSInput { float2 texCoords : TEXCOORD0; }

[shader("fragment")]
float4 fragMain(FSInput input) : SV_Target
{
    float4 color = fxaaPass(input.texCoords);
    if (bloomIntensity > 0.0)
        color.rgb += bloomTex.Sample(bloomSampler, input.texCoords).rgb * bloomIntensity;
    return color;
}
```

> **Note on varying names:** The FXAA vert shader (`screenquad.vert.slang`) emits `texCoords` as `TEXCOORD0`. The original hand-written GLSL used `fragTexCoords`. Slang matches by semantic, not name, so the rename is transparent after compilation.

```slang
// quad.frag.slang
uniform float4 corners;
uniform float  cornerRadius;
uniform float4 color;
uniform float  borderWidth;
uniform float4 borderColor;

struct FSInput
{
    float4 pos          : SV_Position;
    float2 localPosition: TEXCOORD0;
}

[shader("fragment")]
float4 fragMain(FSInput input) : SV_Target
{
    float2 localPos  = (input.localPosition - corners.xy) / (corners.zw - corners.xy);
    float2 quadSize  = corners.zw - corners.xy;
    float2 pos       = localPos * quadSize;
    float2 halfSize  = quadSize * 0.5;

    float2 d    = abs(pos - halfSize) - (halfSize - cornerRadius);
    float  dist = length(max(d, float2(0.0))) - cornerRadius;
    float  alpha = smoothstep(1.0, 0.0, dist);

    if (borderWidth > 0.0)
    {
        float2 innerD    = abs(pos - halfSize) - (halfSize - cornerRadius - borderWidth);
        float  innerDist = length(max(innerD, float2(0.0))) - cornerRadius;
        float  borderMask = smoothstep(1.0, 0.0, dist) - smoothstep(1.0, 0.0, innerDist);
        float  fillMask   = smoothstep(1.0, 0.0, innerDist);
        float4 finalColor = borderColor * borderMask + color * fillMask;
        return float4(finalColor.rgb, finalColor.a * alpha);
    }
    return float4(color.rgb, color.a * alpha);
}
```

> **Dual-source blending** (`layout(location=0, index=1)`) is GLSL-specific. In Slang targeting GLSL 330, use `[[vk::location(0)]]` + `[[vk::index(1)]]` or Slang's `SV_Target0`/`SV_Target1` with a `gl_blend_equation` extension. The safest portable approach for GL 3.3 is to declare two separate `SV_Target` outputs and rely on Slang's GLSL backend handling `index`.

```slang
// text.frag.slang
Texture2D<float4> text        : register(t0);
SamplerState      textSampler : register(s0);
uniform float3    textColor;
```

## Known Special Cases and Gotchas

| Issue | Location | Resolution |
|-------|----------|------------|
| Dual-source blending | `text.frag.slang` | Verify `layout(location=0,index=1)` in output; may need CMake post-process patch if Slang emits `location=1` |
| `transpose(inverse(mat3))` | `pbr.vert.slang` | GLSL 330 supports this; verify Slang emits it correctly for GLSL target |
| `textureSize` → `GetDimensions` | `blur.frag.slang` et al. | `image.GetDimensions(mip, w, h, levels)` — Slang emits `textureSize(image, 0)` for GLSL |
| Varying name mismatch | `fxaa.vert/frag` | Slang matches by semantic not name; `TEXCOORD0` links correctly across stages |
| `SampleOffset` | `fxaa.frag.slang` | Slang → GLSL: emits `textureOffset` — verify in output |
| `alpha` default value | `sprite.frag.slang` | Original GLSL had `uniform float alpha = 1.0;` — Slang uniform defaults are not supported; ensure C++ always sets this uniform |
| `screenquad.vert.slang` compiled twice | `blur.vert.glsl` + `fxaa.vert.glsl` | Both outputs are identical; this is intentional to match existing C++ shader load paths |
